# БекОфисные приключения

Юмористическая обучающая текстовая игра для IT-команд. Ты — руководитель
виртуального продукта: за 10–15 минут нужно провести компанию через рабочий
период, отражая угрозы (кривые релизы, ошибки конфигурации, сбои БД, боссы
вроде «Релиза в пятницу») и не уронив доверие клиентов и коллег.

Лёгкие костыли спасают сейчас и мстят отложенно — выводы про SDLC, тестирование
и оценку рисков игрок делает сам, через последствия своих решений.

## Стек

- **TypeScript + Vite** — статический клиентский сайт, без бэкенда.
- **Supabase** (Postgres + RLS) — учёт игроков и таблица лидеров (опционально).
- **Vitest** — тесты движка и автосимуляция баланса.
- **GitHub Pages** + **GitHub Actions** — хостинг и авто-деплой.

Полный замысел и архитектура — в [PLAN.md](./PLAN.md).

## Разработка

```bash
npm install
npm run dev      # дев-сервер
npm test         # тесты + симуляция баланса
npm run build    # сборка в dist/ (с проверкой типов)
```

## Лидерборд (Supabase)

Игра полностью играбельна без Supabase — таблица лидеров тогда работает локально
(`localStorage`). Чтобы включить онлайн-лидерборд:

1. Применить схему один раз (локально, `DATABASE_URL` с паролем БД — только в `.env`):
   ```bash
   psql "$DATABASE_URL" -f supabase/schema.sql
   ```
   либо вставить содержимое `supabase/schema.sql` в Supabase → SQL Editor.
2. Задать переменные (см. `.env.example`):
   - `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY` — публичные, защита через RLS.

Запись результатов идёт через `SECURITY DEFINER` функцию `submit_run()` с
серверной валидацией, чтение — через вьюшку `leaderboard`. Прямой доступ к
таблицам у анонима закрыт.

> `DATABASE_URL` (пароль БД) и `service_role` ключ в клиент и в репозиторий не
> попадают — только в локальный `.env` / GitHub Actions Secrets.

## Деплой

Пуш в `main` запускает `.github/workflows/deploy.yml`: тесты → сборка → публикация
на GitHub Pages. В настройках репозитория: Pages → Source: **GitHub Actions**.
Ключи Supabase для сборки — в **Settings → Secrets and variables → Actions**.
